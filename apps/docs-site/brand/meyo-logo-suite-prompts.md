# Meyo Logo Suite Prompts

> 当前固定方案是 `Meyo Amber Cats`，母版见 `docs/static/img/brand/meyo-cat-master.png`。下面的 prompt 是历史探索和后续再生成时的参考，不再代表当前项目默认图标。

## Brand Core

- Project name: `Meyo`
- Name origin: `Mengyao -> Meyo`, a private framework name derived from 张梦瑶
- Positioning: `LangGraph-first AgentOS platform`
- Core concepts: `agent runtime`, `knowledge / skill`, `memory OS`, `tool mesh`, `observability`
- Existing visual tone in repo: warm amber, ivory, honey gold, painterly oil texture, quiet premium product feel

## Visual Direction

- Primary direction: `Meyo Amber Cats`
- Meaning: use two adult exotic-shorthair cats as the private brand symbol; the white cat is grumpy and round, the black cat is silly and clear-eyed
- Temperament: warm, artistic, premium, personal, playful but not childish
- Avoid: generic AI symbols, graph nodes, orbit lines, robot head, speech bubble, brain icon, circuit cliché, crypto vibe, purple neon, cluttered cartoon mascot style

## Brand Palette

- Amber: `#D89A2B`
- Honey gold: `#F0B85A`
- Warm ivory: `#FFF2D4`
- Cream brown: `#B98242`
- Ink black: `#11100D`
- Soft pink accent: `#EAA0A0`

## Command Template

If you want to batch-run later with the bundled CLI:

```bash
export CODEX_HOME="${CODEX_HOME:-$HOME/.codex}"
export IMAGE_GEN="$CODEX_HOME/skills/imagegen/scripts/image_gen.py"

python "$IMAGE_GEN" generate-batch \
  --input tmp/imagegen/meyo-logo-suite.jsonl \
  --out-dir output/imagegen/meyo-logo-suite \
  --concurrency 3
```

Path convention:

- `docs/static/img/brand/` keeps the logo files used by the project.
- `docs/brand/` keeps long-lived brand prompt documentation.
- `tmp/imagegen/` is local batch input scratch space.
- `output/imagegen/` is local generated output.
- `tmp/` and `output/` are ignored by git.

Current project note: the active logo set is generated from `docs/static/img/brand/meyo-cat-master.png` into PNG / ICO assets under `docs/static/img/brand/`.
