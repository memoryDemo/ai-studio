# Meyo Brand Assets

这些文件是项目内直接使用的品牌资源，不是临时生成物。

当前方向：`Meyo Amber Cats`。固定母版是两只猫的琥珀色油画图标，后续所有尺寸和组合图都从这个母版派生。

字体方向：带文字版本使用英文艺术手写字标，当前用 macOS 自带 `Snell Roundhand` 风格生成，避免普通技术字体。

## Source

- `meyo-cat-master.png`: 固定母版，不要随便覆盖。
